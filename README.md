# OK to Eat

UK Food Hygiene Ratings for restaurants, takeaways, and food businesses.

**Live:** https://oktoeat.co.uk

## Tech Stack

- React 19 + Vike + vike-photon (SSR)
- Vite
- Hono (API routes)
- Cloudflare Workers + D1

## Development

```bash
npm install
npm run dev
```

## Data

Food hygiene ratings from the [Food Standards Agency](https://www.food.gov.uk/) under the Open Government Licence. ~600,000 UK businesses updated weekly.
